# ASP Example Problem: Simple Path Coloring

## Problem Description
Given three nodes labeled 1, 2, and 3, and two available colors (red and green), color each node so that no two adjacent nodes share the same color.

## Constraints
- Each node must be assigned exactly one color (red or green).
- No two adjacent nodes may share the same color.
- The nodes are connected in a path: node 1 is connected to node 2, and node 2 is connected to node 3.
